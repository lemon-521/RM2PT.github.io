## Version 1.2.1 features:

###  Prerequisite

`Note that JDK8 does not support`
* **[Adopt OpenJDK 11](https://adoptopenjdk.net/)** must be installed to run RM2PT
* **JAVA_HOME** is set to your JDK11 directory
